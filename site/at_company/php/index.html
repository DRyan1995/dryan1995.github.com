1. How to loop through each key/value pair in an array

    <?php
    $age=array("Bill"=>"35","Steve"=>"37","Peter"=>"43");

    foreach($age as $x=>$x_value) {
      echo "Key=" . $x . ", Value=" . $x_value;
      echo "<br>";
    }
    ?>

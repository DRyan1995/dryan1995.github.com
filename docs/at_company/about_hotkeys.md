#Hotkeys Provided By Panpan

- M-x 命令窗口
- C-x C-f 打开文件
- C-J/K 切换标签
- M-m/p 切换窗口（切分后的窗口） 已修改
- C-x 2 水平切分
- C-x 3 垂直切分
- C-x 0 关闭切分窗口
- C-x k 关闭标签
- C-t 在打开的tab中寻找一个tab
- C-M-j 在文件内查找定义（原来的cmd-R）
- C-s 搜索
- C-M-f 全局搜索

- C-</> 跳到航首，行尾

- C-g 退出（相当于esc）
- C-space 开启/关闭选区 （这个快捷键很重要，但是和输入法相冲，你看着办） 已修改
- C-x h 全选
- C-w cut
- M-w copy

- C-y paste
- M-y paste-from-history

- C-/ undo 已修改
- C-? redo
